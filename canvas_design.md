# Directive: Canvas Design

## Goal
Create museum-quality visual art and static designs (posters, art pieces) using a two-step "Philosophy then Expression" approach.

## Context
Standard AI-generated graphics are often generic. This skill ensures artistic depth by first defining a visual philosophy.

## Process
### 1. Design Philosophy Creation
- Define a **movement** (e.g., "Brutalist Joy").
- Articulate the philosophy in 4-6 paragraphs covering space, form, color, and rhythm.
- Emphasize **expert craftsmanship** and minimal text.

### 2. Canvas Expression
- Create a single-page PDF or PNG.
- Use repeating patterns, perfect shapes, and thin fonts.
- Text must be sparse, clinical, and systematic.

## Success Criteria
1. The final piece looks like hours of manual labor by a master.
2. Text is minimal (10%) and design-forward.
3. No elements overlap or fall off the canvas.
